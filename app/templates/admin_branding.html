Outbound communications
Configuration guide ↗
Settings that define who messages come from (sender identity, signatures). The transport layer — SMTP host, Slack webhook, Teams incoming webhook, Twilio — is stored in the notif_channels table and is not yet editable from the UI. Use meridian-nip notif channel add (or hit POST /api/v1/notifications/channels) to register a transport, then reference it from monitor alerts and webhook fan-outs.
