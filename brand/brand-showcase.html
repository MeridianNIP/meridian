USAGE NOTES
· currentColor variants take the CSS color property of their container — drop into any page and set the color once.
· Animated SVG uses inline <style> so it animates standalone in any browser. Not suitable for print / letterhead / email clients.
· Mono variants are intentionally single-color so they reproduce identically on fax, B&W print, and embossed / foiled stamping.
· Minimum legible size: 16×16 px on screen; 10 mm wide in print. Below that, use the center-meridian-only simplified variant (can be generated from mark.svg by removing the rotated ellipses if needed).
· Clear space around the mark: 1/4 of the mark's height on all sides.
